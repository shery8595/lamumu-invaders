# 🐄 Lamumu Invaders 🚀

A fun, cow-themed twist on the classic **Space Invaders** arcade game — built entirely with HTML, CSS, and JavaScript.  
Shoot alien invaders, rack up points, and survive as long as you can while enjoying cosmic cow chaos! 🛸

---

## 🎮 Features
- **Retro pixel art style** using [Press Start 2P](https://fonts.google.com/specimen/Press+Start+2P) font.
- **Smooth animations** for invaders and background cosmic effects.
- **Dynamic levels** with increasing difficulty.
- **Custom sound effects**:
  - 🐄 Moo shooting sound
  - 💥 Breaking and impact effects
- **Powerful UI**: Scoreboard, lives counter, and control hints.
- **Custom themed bullets** (“MOO” letters for bullets).
- **Responsive cosmic background** with animated particles and rings.

---

## 🕹️ Controls
| Key             | Action          |
|-----------------|-----------------|
| `←` / `A`       | Move Left       |
| `→` / `D`       | Move Right      |
| `SPACE`         | Shoot           |
| `P`             | Pause / Resume  |
